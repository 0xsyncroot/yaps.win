# Hướng dẫn cài đặt

## Bước 1: Cài đặt tiện ích

1. Tải từ Chrome Web Store hoặc build từ mã nguồn
2. Mở Chrome → `chrome://extensions/`
3. Bật "Chế độ nhà phát triển"
4. Nhấp chuột "Tải không đóng gói" → Chọn thư mục `dist/`

## Bước 2: Đăng nhập

1. Nhấp chuột vào biểu tượng tiện ích
2. Nhấp chuột "Đăng nhập bằng Telegram"
3. Telegram mở bot → Gửi `/login`
4. Quay lại tiện ích và làm mới

## Bước 3: Cấu hình (Tùy chọn)

1. Tab **Cài đặt**
2. Chọn nhà cung cấp AI
3. Thiết lập độ trễ, số lần thử lại
4. Nhấp chuột "Lưu cài đặt"

---

## Yêu cầu hệ thống

- Chrome 88+ (Manifest V3)
- Kết nối internet
- Tài khoản Telegram
- Tài khoản X (Twitter)
