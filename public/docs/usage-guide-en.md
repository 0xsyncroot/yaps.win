# Usage Guide

## Batch Reply

1. Tab **Tasks**
2. Paste URLs (one URL per line)
3. Click "Add Tasks"
4. Click "Start Processing"

**Example URLs:**
```
https://x.com/user1/status/123456789
https://twitter.com/user2/status/987654321
```

## Newsfeed Auto-Reply

1. Tab **Newsfeed**
2. Select KOLs you want to reply to
3. Configure settings (delay, filters)
4. Click "Start Auto-Reply"
5. Extension will automatically reply to KOL posts in newsfeed

## KOL Monitoring

1. Tab **KOLs**
2. Add KOLs (usernames or URLs)
3. Click "Start Monitoring" for each KOL
4. Extension will monitor and auto-reply to new posts

## Personalization

1. Tab **Account**
2. Click "Rerun Onboarding"
3. Select posts to analyze
4. Extension will create writing style profile
5. Replies will match your writing style
