# Settings

## AI Provider
- **OpenAI**: GPT-4, GPT-3.5
- **Grok**: Grok-4-fast-non-reasoning
- **DeepSeek**: DeepSeek Chat

## Delay Between Replies
Wait time between replies (1-60 seconds)
- **5-10 seconds**: Personal use
- **10-30 seconds**: Commercial use

## Max Retries
Number of retries when reply fails (1-5 times)
- Default: 3 times

## Auto-check Commentable
Automatically check if post is commentable before generating reply
- Should be enabled to avoid wasting quota

## Notifications
- **Notify on Complete**: Notify when all tasks are completed
- **Notify on Error**: Notify when a task fails
